# Relationship Types

Use these relationship labels in LLM Wiki notes.

## Common relationships

- is_part_of
- depends_on
- causes
- supports
- conflicts_with
- leads_to
- requires
- affects
- replaces
- is_example_of
- is_related_to
- is_source_for
- should_update
- should_not_include_by_default
