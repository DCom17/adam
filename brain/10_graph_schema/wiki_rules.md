# LLM Wiki Rules

## Purpose

The LLM Wiki is where AI-compiled structured knowledge lives.

It is not the same as the user's raw thinking.

## Source Routing

Raw notes and pasted material go in:

- `08_sources/raw_articles/`
- `08_sources/raw_transcripts/`
- `08_sources/raw_course_notes/`
- `08_sources/raw_work_docs/`

Compiled pages go in:

- `09_llm_wiki/concepts/`
- `09_llm_wiki/people/`
- `09_llm_wiki/projects/`
- `09_llm_wiki/systems/`
- `09_llm_wiki/decisions/`
- `09_llm_wiki/formulas/`
- `09_llm_wiki/workflows/`
- `09_llm_wiki/maps_of_content/`

## Wiki Note Template

Each wiki note should include:

- Type
- Status
- Confidence
- Last reviewed date
- Definition
- Why it matters
- Related nodes
- Relationship triples
- Source notes
- Review status

## Rule

Do not treat AI-compiled pages as confirmed truth unless reviewed.
