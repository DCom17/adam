# Aliases

**Rules:**
- Rows marked "manual only" in the Notes column are NOT auto-linked.
- Do not add overly generic words as aliases (e.g., "work", "school", "day", "system").
- An alias must unambiguously refer to the canonical node in most contexts.
- Add new aliases conservatively. When in doubt, mark as manual only.
