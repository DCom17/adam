# Monthly Summaries

This folder stores monthly snapshot archives of [[Hunter Tracker]] state.

## Purpose

Monthly summaries capture a point-in-time view of [[Character Level|character level]], stat levels, boss progress, and key wins/misses for the month. They serve as long-term progression records.

## File Naming

Use this format:

```
YYYY-MM_monthly_summary.md
```

Example: `2026-05_monthly_summary.md`

## When to Create

Generate a monthly summary at the end of each month during the [[Weekly Review|weekly review]] closest to month-end, or on the first day of the new month.

Do not create a monthly summary without real data. Do not invent progress.
